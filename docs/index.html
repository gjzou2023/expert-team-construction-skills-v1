全域专家团构建 Skills

一套统一的 AI 专家团构建套系 —— 支持根据用户需求动态构建任意领域专家团，涵盖角色定义、协作流程与交付标准。用户只需触发一次，系统自动完成 8 阶段全流程。

🚀 快速开始

  1. 克隆仓库: git clone https://github.com/gjzou2023/expert-team-construction-skills-v1.git
  2. 选择目标平台（Claude Code / Codex CLI / Hermes / WorkBuddy），将 dist-v2/{platform}/ 下文件复制到对应目录
  3. 触发 team-orchestrator skill，输入你的需求（如"帮我建一个数据分析专家团"）
  4. 系统自动执行 S1→S8 全流程，按提示确认即可
  5. 运行 python import_validate.py 校验完整性

🔧 构建与校验

  1. 修改 skills/全域专家团/ 下的源 SKILL.md 文件
  2. 运行 python build_dist.py 重新构建四平台分发版本
  3. 运行 python import_validate.py 校验完整性
  4. 检查 dist-v2/ 下各平台文件是否最新
